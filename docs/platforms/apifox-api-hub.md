# Apifox API Hub 发布包

## 为什么优先发 Apifox

Apifox 用户天然是 API 开发者，搜索“天气 API”“条码 API”“QQ 头像 API”的意图比普通内容平台更强。

## 准备材料

- OpenAPI 地址：https://apizero.cn/openapi.json
- 官网：https://apizero.cn
- 文档入口：https://apizero.cn/aidocs
- 示例仓库：https://github.com/MageGojo/apizero-examples

## 项目标题

```text
极数本源 ApiZero：免费天气、商品条码、QQ头像、热搜等 API 示例
```

## 项目描述

```text
ApiZero 极数本源提供一组面向开发者的实用 API，包含天气查询、商品条码查询、QQ 头像昵称查询、百度热搜、实时票房、基金估值等接口。

本项目通过 OpenAPI 导入，可直接查看参数、响应字段并进行调试。示例代码见 GitHub：Python / Node.js 均可复制运行。
```

## 标签建议

```text
API, 免费API, 天气API, 商品条码API, QQ头像API, Python, Node.js
```

## 发布步骤

1. 登录 Apifox。
2. 新建项目，选择“导入 OpenAPI”。
3. 输入 `https://apizero.cn/openapi.json`。
4. 项目名使用上面的标题。
5. 描述里放官网和 GitHub 示例仓库。
6. 公开发布到 API Hub。

## 发布后检查

- 搜“天气 API”能否看到项目。
- 随机打开 `weather`、`barcode-lookup`、`qq` 三个接口。
- 确认文档里的示例仓库链接能点击。
