# V2EX 求测帖

V2EX 不适合硬广。语气要像真实开发者求反馈。

## 标题候选

```text
做了一组免费 API 示例，想请大家帮忙测测天气/条码/QQ 头像接口
```

```text
整理了一个 API 示例仓库，包含天气、商品条码、QQ 头像，求反馈
```

## 正文

```text
最近在整理一个面向个人开发者的小 API 示例仓库，先放了几个调用量比较高、比较容易跑通的接口：

- 天气查询：按城市名查实时天气、未来天气
- 商品条码查询：输入条码返回商品名称、品牌、规格、图片
- QQ 信息：输入 QQ 号返回头像、昵称、邮箱
- 百度热搜 / 实时票房 / 基金估值：做内容看板或小项目

仓库地址：
https://github.com/MageGojo/apizero-examples

官网文档：
https://apizero.cn/aidocs

我现在比较想验证两件事：

1. 示例代码复制后是否能直接跑通。
2. 哪些接口对个人项目/小程序/脚本最有用。

如果你刚好做过天气模块、扫码录入、数据看板之类的小项目，欢迎拍砖。尤其想听听哪些接口看起来有用，但实际上不会用。
```

## 评论区回复策略

- 有人反馈失败：先问接口、参数、错误码，不争辩。
- 有人质疑广告：承认是项目求测，强调示例代码开源。
- 有人提需求：记录到 GitHub Issue。
