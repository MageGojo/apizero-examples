# Gitee / GitCode 镜像发布包

## 目的

国内开发者搜索代码时不一定能稳定打开 GitHub。Gitee/GitCode 镜像可以增加国内代码搜索曝光。

## 仓库名称

```text
apizero-examples
```

## 仓库简介

```text
极数本源 ApiZero 可运行示例：天气 API、商品条码查询 API、QQ 头像 API、百度热搜、实时票房、基金估值等 Python / Node.js 示例。
```

## 标签

```text
API, 免费API, 天气API, 商品条码, Python, Node.js
```

## 镜像步骤

```bash
git remote add gitee <你的 Gitee 仓库地址>
git push gitee main
```

或者在 Gitee/GitCode 后台选择“从 GitHub 导入仓库”。
