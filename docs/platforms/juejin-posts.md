# 掘金发布包

掘金读者比 CSDN 更挑标题和代码质量。标题要像工程实践，不要像广告。

## 首发 3 篇

### 1. Node.js 调天气 API，不申请高德/百度 Key

标题：

```text
Node.js 做天气查询接口：不用申请高德/百度 Key，直接按城市名返回天气
```

摘要：

```text
本文用 Node.js 18 原生 fetch 调用天气 API，实现按城市名查询实时天气、未来天气和天气摘要。适合小程序后端、站点天气卡片和定时任务。
```

正文使用：`docs/csdn/04-nodejs-weather-api.md`
