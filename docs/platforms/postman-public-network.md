# Postman Public API Network 发布包

## 为什么发 Postman

Postman 的用户是 API 调试和集成用户。即使中文流量不如 CSDN，信任度和国际曝光更好。

## Workspace 名称

```text
ApiZero Examples
```

## Workspace 描述

```text
Runnable ApiZero API examples for weather, barcode lookup, QQ profile info, Baidu hot search, movie box office, and fund estimate APIs.

Docs: https://apizero.cn/aidocs
Examples: https://github.com/MageGojo/apizero-examples
OpenAPI: https://apizero.cn/openapi.json
```

## Collection / API 描述

```text
ApiZero provides practical HTTP APIs for developers. This public workspace focuses on examples that are easy to test: weather API, barcode lookup API, QQ avatar API, and content data APIs.
```

## 发布步骤

1. 登录 Postman。
2. 创建 Public Workspace：`ApiZero Examples`。
3. 导入 OpenAPI：`https://apizero.cn/openapi.json`。
4. 在 Workspace 描述中放 GitHub 示例仓库。
5. 将 Workspace 设置为 Public。
6. 在 GitHub README 里补 Postman 链接。

## 建议置顶接口

- `GET /api/weather`
- `GET /api/barcode-lookup`
- `GET /api/qq`
- `GET /api/hot-baidu`
- `GET /api/movie-box`
- `GET /api/fund`
