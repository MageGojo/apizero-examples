## 发布小规则

- 每天最多发 1-2 篇，不要一次性全发。
- 标题不要改成品牌广告，保留搜索词。
- 每篇发布后记录阅读量、收藏、评论和官网点击。
- 阅读低于 20 的同类标题先暂停，换接口或换场景。
