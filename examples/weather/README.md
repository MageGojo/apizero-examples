# 免费天气 API 示例

本目录包含两个天气方向：

- `weather`：彩云天气接口，当前站内调用量最高，适合多数天气模块。
- `moji-weather`：墨迹天气接口，字段更完整，适合需要实况、AQI、生活指数的场景。

## 运行

```bash
python examples/weather/python_weather.py
node examples/weather/node_weather.mjs
python examples/weather/python_moji_weather.py
```

## 文档

- 彩云天气：https://apizero.cn/aidocs/weather
- 墨迹天气：https://apizero.cn/aidocs/moji-weather
