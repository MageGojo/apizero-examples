# 商品条码查询 API 示例

本目录包含两个条码接口：

- `barcode-lookup`：免费版，适合扫码补全商品名称、品牌、规格、图片。
- `barcode-gs1`：PRO 版，适合需要官方注册主体、上市日期、厂商信息的场景。

## 运行

```bash
python examples/barcode/python_barcode_lookup.py
node examples/barcode/node_barcode_lookup.mjs
python examples/barcode/python_barcode_gs1.py
```

## 文档

- 免费版：https://apizero.cn/aidocs/barcode-lookup
- PRO 版：https://apizero.cn/aidocs/barcode-gs1
